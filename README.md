# remove-baidu-hot-list-plugin

## Project request

```
需求：以按钮的形式隐藏百度热榜。
缺点：需要手动点击按钮来确认对单个页面的隐藏操作，无全局操作，无状态记录。
```

## Project technology stack

```
Chrome 插件 + Vue
```

## Plugin usage

```
1、拉取代码。
2、安装依赖。
3、执行构建。
4、在谷歌浏览器中找到扩展程序设置，打开开发者模式。
5、点击"加载以解压的扩展程序"，选择刚刚构建后的代码。
6、接下来就是使用正常插件的步骤。
```
